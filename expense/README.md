# Personal Expense Analysis Dashboard

This project analyzes spending behavior using Python, Pandas, and Matplotlib to uncover patterns, trends, and insights.

## Key Features
- Data cleaning and preprocessing
- Category-wise expense analysis
- Spending trend visualization
- Insights and recommendations

## Tools Used
Python, Pandas, Matplotlib, Jupyter Notebook

## Insights
Bills were identified as the largest expense category, with spending spikes driven by fixed costs. The analysis highlights opportunities to improve budgeting and financial awareness.

## Project Goal
To practice exploratory data analysis and demonstrate how data can support better decision-making.
